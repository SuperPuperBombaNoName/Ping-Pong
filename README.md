## 🏓 Ping-Pong AI Edition
## Классическая аркада на Python & Pygame
Это современная реализация легендарной игры "Пинг-Понг", написанная на Python. Проект демонстрирует базовые принципы разработки 2D-игр: работу с кадровой частотой, физику столкновений и обработку пользовательского ввода.
------------------------------
## 🚀 Особенности

* Два режима игры: Соревнование для двух игроков за одной клавиатурой.
* Плавная анимация: Оптимизированный игровой цикл с частотой 60 FPS.
* Динамическая сложность: Ускорение мяча после каждого успешного удара.
* Интерфейс: Автоматический подсчет очков и визуальное разделение поля.

------------------------------
## 🛠 Технологический стек

* Язык: Python 3.x
* Библиотека: Pygame (обработка графики и событий)

------------------------------
## 🎮 Управление

| Игрок | Вверх | Вниз |
|---|---|---|
| Левый игрок | W | S |
| Правый игрок | Arrow Up | Arrow Down |


## 🔧 Логика разработки (для разработчика)

   1. Инициализация: Настройка окна (например, 800x600) и цветов (RGB).
   2. Объекты: Создание классов или структур для мяча и ракеток на основе pygame.Rect.
   3. Коллизии: Использование метода .colliderect() для определения момента удара мяча о ракетку.
   4. Границы: Проверка координат мяча для отскока от верхней/нижней стен и фиксации гола.
